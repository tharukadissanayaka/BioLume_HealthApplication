BioLume: A Comprehensive Health & Wellness Tracking Application
---------------------------------------------------------------
BioLume is a full-stack health and wellness tracking application that empowers users to monitor and manage their daily health metrics. The application provides real-time insights into activity, hydration, sleep patterns, and overall wellness metrics through an intuitive dashboard
